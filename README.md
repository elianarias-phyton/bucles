# Ejercicios de Lógica con Python - Bucles y Control de Flujo

Este repositorio contiene una colección de **20 ejercicios progresivos** diseñados para practicar estructuras de repetición en Python. 

Los ejercicios están organizados en **6 niveles de dificultad**, comenzando con bucles simples y avanzando hacia lógica más compleja con validaciones, acumuladores y pequeños programas interactivos.

## 🚀 Contenido por Niveles

* **Nivel 1:** Introducción a bucles `for`, impresión repetitiva y patrones simples con texto y asteriscos.
* **Nivel 2:** Uso de **acumuladores**, sumatorias y generación de tablas de multiplicar.
* **Nivel 3:** Validación de datos ingresados por el usuario y cálculo de promedios.
* **Nivel 4:** Uso de bucles `while` con **condiciones de salida** y contadores.
* **Nivel 5:** Simulación de situaciones reales (registro de alumnos, ahorro y control de stock).
* **Nivel 6:** Ejercicios más completos como **patrones de figuras** y un **juego de adivinanza** con números aleatorios.

## 🛠️ Conceptos Aplicados

* Entrada de datos con `input()` y conversión de tipos (`int`, `float`).
* Uso de bucles `for` y `while`.
* Gestión de **acumuladores y contadores**.
* Validación de rangos y condiciones.
* Generación de patrones con texto (`*`).
* Uso de la librería `random`.
* Control de flujo con `break`.

## 💻 Cómo ejecutar

Cada ejercicio se encuentra **comentado dentro del archivo principal**.

Para probar uno:

1. Selecciona el bloque del ejercicio.
2. Elimina el `#` para descomentarlo.
3. Ejecuta el programa desde PyCharm.

Atajo útil en PyCharm:

```
Ctrl + /
```

Esto permite **activar o desactivar rápidamente los ejercicios**.
